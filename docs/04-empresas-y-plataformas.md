# Doc 04 — Empresas y plataformas, priorizadas por país

Lista accionable. Cada bloque dice **a quién aplicar, dónde, en qué orden, con qué expectativa de salario y qué cuidar**.

Convención de prioridad:
- **P0** = aplicar esta semana. Match alto, alta probabilidad de respuesta.
- **P1** = aplicar dentro de 30 días. Match razonable, requiere algo de pulido.
- **P2** = oportunista, solo si hay vacante específica que calce.
- **P3** = aspiracional, no quemar pólvora ahí todavía.

## 🇺🇸 USA-direct — aspiracional (P3 hasta inglés C1+)

No quemar pólvora ahí todavía. Cuando llegues a C1 funcional + 6 meses de track record remoto:

- **Wellfound** (ex AngelList) — startups Series A–C, filtro `remote LATAM friendly`.
- **WeWorkRemotely** — filtro fuerte (70% requieren US/EU residency).
- **RemoteOK / Remotive** — similar.
- **Toptal** — proceso de admisión exigente, pero buenísimo si pasas. Espera a tener inglés sólido.
- Outreach directo a CTOs/founders por LinkedIn DM (alta conversión cuando se hace bien).

## 💼 Freelance — canal paralelo desde día 1

| Plataforma | Mejor uso | Cuánto cobrar al inicio |
|---|---|---|
| **Workana** | Clientes LATAM + España. Acumular 2–3 reviews 5 estrellas rápido. | 15–25 USD/h al inicio para acumular reviews; subir después. |
| **Upwork** | Clientes globales. Más difícil sin reviews; entras agresivo. | 18–25 USD/h primer trimestre; 30–45 USD/h después. |
| **Contra** | Mejor para diseño/marketing que dev, pero hay nicho. | A discreción. |
| **Codementor** | Tutoring + side income. | 20–40 USD/h. |
| **Toptal / Lemon.io / Arc.dev** | Mid-senior con inglés C1+. Esperar Etapa 3. | 50+ USD/h. |
| **HackerOne / Bugcrowd / Intigriti** | Bug bounty, ingreso variable. Complemento, no canal principal. | Por hallazgo. |
